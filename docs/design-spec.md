# UI 设计规范 — MP3 听力播放器

## 设计原则

- **简洁至上**：每个屏幕只展示必要信息，避免视觉干扰
- **触控优先**：所有可点击区域最小 44×44px（符合移动端触控标准）
- **高对比度**：黑色背景 + 白色文字，确保在各种光线条件下可读

## 配色方案

| 用途 | 色值 | 说明 |
|------|------|------|
| 页面背景 | `#000000` | 纯黑 |
| 主文字 / 图标 | `#FFFFFF` | 纯白 |
| 次要文字 | `#B0B0B0` | 浅灰（文件名、时间等辅助信息） |
| 主点缀色 | `#64B5F6` | 淡蓝（按钮激活、进度条、当前选中） |
| 按钮背景 | `#1E1E1E` | 深灰（未选中按钮底色） |
| 按钮边框 | `#333333` | 中灰（分隔线、按钮边框） |

## 字体

- **系统默认无衬线字体**：`system-ui, -apple-system, sans-serif`
- 标题：`20px bold`
- 正文：`16px normal`
- 时间显示：`14px monospace`（等宽字体，数字对齐更好看）

## 布局（手机竖屏优先，`max-width: 480px`）

```
┌──────────────────────────────────┐
│  📁 选择文件夹     [文件夹名称]    │  ← 顶部工具栏
├──────────────────────────────────┤
│                                  │
│    [当前文件名.mp3]              │  ← 文件名显示
│                                  │
│         ⏪  ▶️/⏸  ⏩             │  ← 播放控制按钮（大号）
│                                  │
│  ──●─────────────────────────   │  ← 进度条
│  01:23               03:45       │  ← 当前时间 / 总时长
│                                  │
│  [后退 2s] [4s] [8s] [16s]     │  ← 快退步长选择
│  [快进 2s] [4s] [8s] [16s]     │  ← 快进步长选择
│                                  │
│  0.5x │ 0.75x │ 1x │ 1.5x      │  ← 倍速切换
│                                  │
├──────────────────────────────────┤
│  📄 文件列表（可滚动）            │
│  ├─ 听力01.mp3                  │  ← 文件列表
│  ├─ 听力02.mp3                  │
│  └─ 听力03.mp3                  │
└──────────────────────────────────┘
```

## 按钮状态

| 状态 | 样式 |
|------|------|
| 默认 | 深灰背景 `#1E1E1E`，白字 |
| 悬停 | 边框变淡蓝 |
| 激活/选中 | 淡蓝背景 `#64B5F6`，黑字 |
| 播放中 | 播放按钮显示淡蓝光圈动画 |

## 响应式适配

- **手机竖屏**（< 480px）：单列布局，按钮占满宽度
- **手机横屏 / 平板**（480px - 768px）：按钮两列排列
- **桌面**（> 768px）：居中显示，最大宽度 600px，两边留白
