# 需求文档 — MP3 听力播放器

## 项目背景

用户为英语四级备考者，需要在手机（荣耀 400）和电脑（Windows 11）上播放下载的 MP3 听力文件，核心诉求是方便地控制播放速度和精准跳转。

## 用户角色

- **主要用户**：英语学习者，非技术背景
- **使用场景**：日常听力练习、精听训练

## 功能需求

### F1 — 文件夹选择
- 用户可以手动选择一个文件夹
- 软件自动扫描该文件夹内所有 `.mp3` 文件
- 显示文件列表供用户选择播放
- 关闭后重新打开，自动恢复上次选择的文件夹

### F2 — MP3 播放
- 点击文件开始播放
- 显示当前播放文件名

### F3 — 播放控制
- **播放/暂停**：一键切换
- **倍速播放**：支持 0.5x、0.75x、1x、1.5x 四档
- **快进/后退**：按固定步长跳转，步长可选 2 秒、4 秒、8 秒、16 秒
- **进度条**：可拖拽/滑动，精确定位到任意秒数

### F4 — 设置记忆
- 记住用户上次选择的播放速度
- 记住用户上次选择的快进/后退步长
- 记住上次播放的文件和进度位置

## 非功能需求

### N1 — 跨平台
- Android（Chrome 浏览器）
- Windows 11（Chrome / Edge 浏览器）

### N2 — 安装方式
- 浏览器打开即用，无需安装
- 支持 PWA「添加到主屏幕」，体验接近原生 APP

### N3 — UI 要求
- 黑色背景 + 白色 UI 元素
- 淡蓝色作为点缀色（按钮高亮、进度条等）
- 简洁直观，按钮足够大方便手指操作
- 同时适配手机竖屏和电脑宽屏

## 不包含的功能（明确排除）

- 不连接网络，不获取在线资源
- 不需要播放列表/歌单功能
- 不需要后台播放（切换应用后继续播放）
- 不需要耳机线控
