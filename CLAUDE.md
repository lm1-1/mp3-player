# CLAUDE.md — MP3 听力播放器项目指引

## 项目简介

这是一个面向英语四级备考者的跨平台 MP3 播放器（PWA 网页应用）。用户为**非技术背景**，所有开发工作需保持简洁、稳定、安全。

## 工作规范

### 通用原则
1. **逐步推进**：每个阶段完成后，记录日志，等待用户确认再进入下一阶段
2. **先读后改**：编辑文件前必须先读取当前内容
3. **代码简洁**：不引入不必要的第三方依赖，不提前做「可能用得着」的功能
4. **安全第一**：不上传文件到任何外部服务器，所有操作本地完成
5. **中文交流**：所有面向用户的文字使用中文

### 每次修改代码后
- 更新当日开发日志（`devlogs/YYYY-MM-DD.md`）
- 标记完成事项和待办事项

### 每日开始工作时
- 读取当日开发日志，了解当前进度和待办

## 关键设计决策

- **PWA 而非原生 APP**：用户要求浏览器打开即用，无需安装
- **File System Access API**：用于读取用户本地 MP3 文件夹
- **HTML5 Audio**：浏览器原生音频播放，无需额外库
- **无后端**：纯前端应用，所有数据存储在浏览器本地（localStorage + IndexedDB）
